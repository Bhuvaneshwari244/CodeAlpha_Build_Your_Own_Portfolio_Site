## Description
This is a personal portfolio website built for **Task 3** of the **CodeAlpha Internship**. It serves as a central hub to showcase my skills, resume, and the projects I have built during this internship.

## Features
- **Responsive Design:** Mobile-friendly navigation with a hamburger menu.
- **Smooth Scrolling:** Seamless navigation between sections (Home, About, Skills, Projects, Contact).
- **Project Showcase:** Highlights recent work with descriptions and tags.
- **Interactive UI:** Hover effects on skill cards and buttons.

## Technologies Used
- HTML5
- CSS3 (Flexbox & Media Queries)
- JavaScript (Mobile Menu Toggle)
- FontAwesome (Icons)

## How to Run
1. Download the source code.
2. Open `index.html` in a browser.
3. (Optional) Hosted live via GitHub Pages.
